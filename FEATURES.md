### How to Use

#### From Selected Text
1. Highlight any text on a webpage
2. Right-click the selection
3. Choose "Add [text] to Vikunja"
4. The popup opens with the text as the task title

#### From Links
1. Right-click any link on a webpage
2. Choose "Add link to Vikunja"
3. The popup opens with the link text/URL as the task

### Benefits
- Capture information without switching context
- Faster task creation from web content
- Automatic source attribution

## üïê Due Date Shortcuts

### Available Shortcuts
- **Today**: Sets due date to today at 11:59 PM
- **Tomorrow**: Sets due date to tomorrow at 11:59 PM
- **Next Week**: Sets due date 7 days from now at 11:59 PM
- **Clear**: Removes the due date

### How to Use
1. Click any shortcut button below the due date field
2. The datetime picker updates automatically
3. You can still manually adjust the time if needed

### Benefits
- Eliminates repetitive date picking
- Consistent end-of-day times
- Faster task creation

##  Tips & Tricks

### Keyboard Shortcuts
- **Tab**: Navigate between fields quickly
- **Enter** (in label field): Add label or create new one
- **Enter** (in form): Submit the task (when not in a textarea)


### Workflow Suggestions
1. **Set up favorites**: Star your main projects once
2. **Use context menus**: Capture ideas as you browse
3. **Quick dates**: Use shortcuts for 90% of due dates
4. **Label consistently**: Type common labels quickly
5. **Create labels on-the-fly**: No need to pre-create all labels

### Best Practices
- Create common labels like "bug", "feature", "idea", "urgent"
- Favorite your active projects
- Use "Next Week" for non-urgent items
- Clear description field for quick captures
- Use label search to quickly find and apply existing labels
